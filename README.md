# Steins-Gate-Clock

Inspired by the Steins;Gate anime.

## Features

- **Microseconds Toggle**:
  - Press `M` to display microseconds.
  - Press `N` to hide microseconds.

- **Border Toggle**:
  - Press `W` to remove borders around the clock.
  - Press `Q` to add borders around the clock.
